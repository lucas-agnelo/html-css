Representando Cores

Por nome

Para representar cores com nome podemos utilizar a declaração de color ou background-color com o nome da cor, o que é bastante limitado. Exemplo: color: blue;

Por Hexadecimal

Para representar cores com nome podemos utilizar a declaração de color ou background-color com Hexadecimal, com +65 milhões de possíveis cores. Exemplo: color: #0000ff;

Por RGB - Red, Green e Blue - (e Alpha para transparência)

Para representar cores com nome podemos utilizar a declaração de color ou background-color com RGB e RGBa, com +65 milhões de possíveis cores. Exemplo: color: rgb(0, 0, 255);

Por HSL - Hue, Saturation e Luminosity - (e Alpha para transparência)

Para representar cores com nome podemos utilizar a declaração de color ou background-color com HSL, com +65 milhões de possíveis cores. Exemplo: color: hsl(240, 100%, 50%);
