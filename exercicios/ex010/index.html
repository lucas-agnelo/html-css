Trabalhando com Links

Links são escritos da seguinte forma:

<a href="ulr">Texto do link</a>

Links Externos

Para criar links externos, ou seja, que direcionam para fora do seu site, deve-se utilisar os parâmetros target="_blank" para abrir em outra aba e rel="external" para alertar mecanismos de busca que é um link externo. Por Exemplo:

Esse é um link para o Google.

Esse é um link para o Repositório do Curso.

Links Internos

Links internos direcionam para páginas dentro do mesmo servidor, e NÃO DEVEM utilizar o parâmetro target="_blank", além de ser interessante que contenham rel="next" ou rel="prev" para criar uma navegação interna que possa ser entendida pelos mecanismos de busca.

Esse é um link para a segunda página.

Esse é um link para a terceira página.

Links de Download

Para criar links de download devemos preencher o href com o caminho do arquivo, e adicionar os parâmetros download="nome-do-arquivo.ext" e type="application/tipo-da-aplicacao-encontrada-no-iana-org"
