Outras Formatações

Código-Fonte e Pré-formatação

Se eu quiser avisar que em CSS posso mudar a cor do body com o seguinte código: body { background-color: blue; } você pode usar a tag CODE.

No entanto, o código não fica indentado. Para fazer um texto seguir a exata formação com que foi escrita no código, incluindo espaços, utilize a tag PRE.

        
body {
    background-color: green;
}

Citações

Citações Simples

Ao invés de Utilizar aspas simples, que dever ser utilizadas apenas em expressões que não são citações, utilizamos a tag Q. Ela dá o sentido semântico de se está citando alguém. Por exemplo, o professor de HTML vive citando que o pai de uma amigo diz que: O computador é um burro muito rápido.

Citações Completas

Para citações completas, retiradas por exemplo de um site ou livro, você pode utilizar a tag BLOCKQUOTE. Ela criará uma citação separada, que através do parâmetro "cite" permite que avise de onde vem aquela citação. Exemplo:

When you first start learning how to hack Wi-Fi, it can be tempting to use your powers on Wi-Fi networks you don't have permission to use. Needless to say, this can get you into a lot of trouble because Wi-Fi hacking isn't particularly subtle, so it's easy to get caught. To keep you from getting into a predicament, we want to make sure you have access to a couple of fun Wi-Fi hacking games.

Abreviações

Caso queira deixar o significado de uma sigla para quando alguém passar o mouse sobre ela, você pode usar a tag ABBR e preencher o parâmetro title com o significado como utilizo aqui em HTML e CSS.
