Anotações

Quando queremos mecher em um elemento filho de outro, isso é, aninhado dentro do outro, nós utilizamos o sinal de <, como no exemplo abaixo em que queremos configurar um <p> que esta dentro de uma <div>:


        div > p {
            declarações...
        }

Usei para criar três parágrafos escondidos em cada div, usando também a pseudo-classe :hover - Exemplo do cógigo abaixo:


        div > p {
            display: none;
        }

        div:hover > p {
            display: block;
            background-color: #000;
            color: white;
        }
