Parágrafos e Quebras de Linhas

Isso é um parágrafo de teste para mostrar que não importa como se organiza o texto, o que manda em um página na web são as marcações do HTML

Toda vez que quiser iniciar um novo parágrafo, você precisará utilizar as tags de abertura e fechamento <p> e </p>.

Caso você queira criar uma quebra de linha no meio de um mesmo parágrafo, você deve utilizar a tag <br>. Ela criará
uma quebra de linha igual a essa.

É importante lembrar que a tag <br> não possui uma tag de fechamento.

Existem ainda muitos outros caracteres especiais para serem utilizados em HTML. Por exemplo:
Trade Mark: ™
Copyright: ©
Registrado: ®
Ou setas: ↑ ↓ ← →
