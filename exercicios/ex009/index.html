Trabalhando com Listas

Listas Ordenadas

Para criar listas ordenadas utilizamos a tag OL, que aceita os parêmatros "type" para definir o tipo de lista, e "start" para definir apartir de onde a lista deve começar. Exemplos:

OL - type="1" - default

  1. Acordar
  2. Tomar banho e escovar dentes
  3. Tomar café
  4. Meditar

OL - type="a"

  1. Acordar
  2. Tomar banho e escovar dentes
  3. Tomar café
  4. Meditar

OL - type="I" e start="2"

  1. Acordar
  2. Tomar banho e escovar dentes
  3. Tomar café
  4. Meditar

Listas não Ordenadas

Para criar listas não ordenadas utilizamos a tag UL, que aceita o parâmetro "type". Exemplos:

UL - type="disc" - default

  • Pão
  • Leite
  • Manteiga
  • Café

UL - type="circle"

  • Pão
  • Leite
  • Manteiga
  • Café

UL - type="square"

  • Pão
  • Leite
  • Manteiga
  • Café

Listas de Definição

Para criar uma lista de definição utilizamos a tag DL (definition list), que deve conter um DT para o termo, e um DD para a descrição do termo.

DL - DT - DD

HTML
Linguagem de marcação para criação de conteúdo de um site - foco na semântica.
CSS
Linguagem de marcação para criação de Design de um site - foco na forma.
Javascript
Linguagem de programação para ciração de interatividade em um site

Anotações da Aula:

Se quiser alterar mais de uma tag ao mesmo tempo, segure ALT e clique em cada tag, então ao editar, editará todas ao mesmo tempo.
