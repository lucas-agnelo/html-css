Anotações

Quando quiser estilizar mais de uma classe ao mesmo tempo, utilizar:


        h1, h1 {
            font-family: declarações;
        }

O W3C recomenda o uso de px como tamanho absoluto de fonte, e de em como tamnha relativo de fonte.

1em geralmente é igual a 16px, e é relativo ao tamanho das letras maiúsculas da fonte.

Configuração de Fonte

Nem todas as fontes possuem todas essas propriedades, mas essas são as configurações comuns.


        font-style: italic;
        font-weight: lighter / normal / bold / bolder ou 100-900;
        font-size: 16px ou 1em;
        font-family: Arial, sans-serif;

Shorthand para configuração de fonte

font: style weight size family;
